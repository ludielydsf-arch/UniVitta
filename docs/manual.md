# Manual do Usuário

## Acesso
- Equipe: entrar via página de Login e usar menus para módulos.
- Paciente: acessar Portal, realizar login e visualizar agendamentos.

## Módulos
- Pacientes/Médicos: cadastrar, editar, excluir e listar.
- Agendamento: escolher visualização, navegar pelas datas e criar compromissos.
- Prontuário: registrar anamnese, prescrição e anexar exames.
- Financeiro: registrar pagamentos e acompanhar tabela.
- Estoque: inserir itens e monitorar alertas de mínimo/validade.
- Relatórios: conferir KPIs e exportar CSV de pagamentos.
- Comunicação: usar chat interno em tempo real.

## Dicas
- Use o campo de busca do navegador para encontrar registros nas tabelas.
- Na página de Agendamento, utilize as setas para navegar entre períodos.
- Em Prontuário, selecione o paciente antes de enviar arquivos.
